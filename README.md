# React Firebase Chat Example

## Features

- uses:
  - React (create-react-app)
  - Firebase
  - one to one
  - authentication
  - group chat

## Installation

- `git clone`
- `cd react-firebase-chat`
- `npm install`
- `npm start`
- visit http://localhost:3000/
- Use your own Firebase Credentials

### Use your own Firebase Credentials

- visit https://firebase.google.com/ and create a Firebase App
